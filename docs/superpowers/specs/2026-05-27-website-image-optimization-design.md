# 网站配图全面优化设计方案

## 项目概述

### 背景
美容医学知识博客（Hugo + PaperMod主题）需要全面优化配图，提升网站视觉质量和内容可信度。当前图片存在质量问题，需要从网络搜索高质量、专业的配图。

### 目标
1. 优化所有9篇文章的featuredImage和card图片
2. 优化文章内关键章节配图
3. 建立统一的图片规范标准
4. 提升网站整体视觉质量和用户体验

## 图片规范标准

### 尺寸规范
- **featuredImage**：1200×630像素（16:9比例，适合社交媒体分享）
- **card图片**：800×450像素（16:9比例，适合列表展示）
- **文章内配图**：1000×600像素（5:3比例，适合内容展示）

### 格式要求
- 格式：JPEG（照片类）或PNG（图形类）
- 质量：JPEG 85%以上，PNG-24
- 文件大小：单张图片不超过500KB

### 风格指南
- **指南类文章**：专业医学风格，使用高质量医学插图、解剖图、手术示意图
- **新闻类文章**：现代简约风格，使用医疗设备、诊室、专业团队照片
- **趋势类文章**：时尚美容风格，使用社交媒体界面、趋势图表、流行元素

## 图片来源和搜索策略

### 主要来源（免费商用）
1. **Unsplash**：高质量免费图片，适合美容、医学主题
2. **Pexels**：免费商用图片，有专门的医学和美容分类
3. **Pixabay**：免费图片和矢量图，适合医学插图
4. **医学专业图库**：如MedicalRF、Science Photo Library（必要时使用）

### 搜索关键词策略
- **指南类**：blepharoplasty, rhinoplasty, injectable, dermal filler, botox, medical procedure
- **新闻类**：aesthetic clinic, medical team, cosmetic surgery, beauty technology
- **趋势类**：social media trends, beauty influencer, cosmetic trends, xiaohongshu

### 质量筛选标准
- 分辨率至少1200像素宽
- 无水印、无版权问题
- 与文章内容高度相关
- 专业、可信的视觉效果

## 技术实现细节

### 图片处理流程
1. **搜索下载**：使用webfetch工具从免费图库搜索并下载图片
2. **尺寸调整**：使用ImageMagick或类似工具调整尺寸
3. **质量优化**：压缩图片，确保文件大小<500KB
4. **格式转换**：统一转换为JPEG或PNG格式
5. **重命名**：按照规范重命名图片文件

### Hugo配置更新
- 更新每篇文章的frontmatter中的featuredImage路径
- 确保card图片路径正确
- 测试图片在Hugo中的显示效果

### 质量保证
- 每张图片必须与文章内容高度相关
- 图片必须专业、可信，符合医学美容主题
- 避免使用AI生成的假图或低质量图片
- 确保图片版权合规，可免费商用

## 风险评估和应对

### 潜在风险
1. **图片版权问题**：确保所有图片来自免费商用图库
2. **图片质量不一致**：建立统一的质量筛选标准
3. **图片与内容不相关**：仔细筛选，确保高度相关
4. **技术实现困难**：准备备用方案和工具

### 应对措施
1. **版权验证**：只使用明确标注可免费商用的图片
2. **质量检查**：建立多级质量检查机制
3. **内容匹配**：与文章作者确认图片相关性
4. **技术准备**：准备多种图片处理工具和方案
